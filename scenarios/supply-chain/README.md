# Supply Chain API

## Description

Suppliers, products, warehouses, inventory, purchase orders, shipments

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Supplier management
- Product catalog
- Warehouse operations
- Inventory tracking
- Purchase order management
- Shipment tracking
