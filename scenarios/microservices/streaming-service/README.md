# Streaming Service API

## Description

Video streaming and playback management for StreamFlix platform. Handles stream initialization, quality selection, subtitle/audio tracks, downloads for offline viewing, playback state tracking, and DRM/license management.

## Available Specifications

This microservice includes API specifications in the following formats:

- **OpenAPI 3.0** - `openapi/json/` and `openapi/yaml/`
- **RAML 1.0** - `raml/`

## API Features

- Stream initialization and playback URL generation
- Quality/bitrate selection and switching
- Subtitle track management (selection, availability)
- Audio track management (language, availability)
- Download management for offline viewing
- Playback state and progress tracking
- DRM license acquisition and management
- Streaming manifest generation (HLS/DASH)
- Adaptive bitrate selection
