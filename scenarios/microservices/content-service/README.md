# Content Service API

## Description

Content catalog management for the StreamFlix platform. This microservice manages movie and TV show catalogs, including content metadata, search and filtering, genre and category management, ratings and reviews, and regional availability.

## Available Specifications

This microservice includes API specifications in the following formats:

- **OpenAPI 3.0** - `openapi/json/` and `openapi/yaml/`
- **RAML 1.0** - `raml/`

## API Features

- Content catalog operations (movies, TV shows, episodes)
- Full-text search with advanced filtering
- Genre and category management
- Content metadata (cast, crew, ratings, reviews)
- Content availability by region
- Content ratings and review management
