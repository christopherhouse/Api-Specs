# Billing Service API

## Description

Subscription management, payment processing, invoicing, and billing cycle management for the StreamFlix platform

## Available Specifications

This microservice includes API specifications in the following formats:

- **OpenAPI 3.0** - `openapi/json/` and `openapi/yaml/`
- **RAML 1.0** - `raml/`

## API Features

- Subscription plans management (basic, standard, premium)
- Subscription lifecycle (create, upgrade, downgrade, cancel)
- Payment method management (add, update, delete)
- Invoice and payment history tracking
- Billing cycle management
- Promotional codes and discount application
- Subscription renewal and cancellation handling
- Payment receipt generation
