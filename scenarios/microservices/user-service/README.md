# User Service API

## Description

User authentication, profile management, preferences, and watch history for StreamFlix platform

## Available Specifications

This microservice includes API specifications in the following formats:

- **OpenAPI 3.0** - `openapi/json/` and `openapi/yaml/`
- **RAML 1.0** - `raml/`

## API Features

- User authentication (login, logout, token refresh)
- User registration and account management
- Profile management (multiple profiles per account)
- User preferences and settings
- Watch history tracking
- Watchlist management
