# Recommendation Service API

## Description

Personalized content recommendations engine for the StreamFlix platform providing intelligent suggestions based on user behavior, preferences, and viewing patterns.

## Available Specifications

This microservice includes API specifications in the following formats:

- **OpenAPI 3.0** - `openapi/json/` and `openapi/yaml/`
- **RAML 1.0** - `raml/`

## API Features

- Personalized recommendations based on watch history
- Trending and popular content recommendations
- Similar content suggestions
- "Because you watched X" contextual recommendations
- Genre-based recommendations
- User-specific recommendation preferences
- Recommendation feedback (thumbs up/down)
- Recommendation filtering and sorting
