# Procurement API

## Description

Vendors, contracts, requisitions, purchase orders, approvals, invoices

## Available Specifications

This domain includes API specifications in the following formats:

- **WADL** - `rest/wadl/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Vendor management
- Contract administration
- Requisition processing
- Purchase order creation and tracking
- Approval workflows
- Invoice handling
