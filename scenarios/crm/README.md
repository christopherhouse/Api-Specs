# CRM API

## Description

Accounts, contacts, opportunities, activities, products, quotes

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **WADL** - `rest/wadl/`
- **WSDL 1.1 (SOAP)** - `soap/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Account management
- Contact tracking
- Opportunity pipeline
- Activity logging
- Product catalog
- Quote generation
