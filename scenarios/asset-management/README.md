# Asset Management API

## Description

Assets, categories, locations, work orders, audits

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **WADL** - `rest/wadl/`
- **WSDL 1.1 (SOAP)** - `soap/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Asset tracking and categorization
- Location management
- Work order processing
- Audit trail capabilities
