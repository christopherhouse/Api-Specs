# Fleet Management API

## Description

Vehicles, drivers, trips, maintenance records, fuel logs

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **WADL** - `rest/wadl/`
- **WSDL 1.1 (SOAP)** - `soap/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Vehicle tracking and management
- Driver assignment and monitoring
- Trip logging
- Maintenance scheduling and records
- Fuel consumption tracking
