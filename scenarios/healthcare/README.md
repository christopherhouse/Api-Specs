# Healthcare API

## Description

Patients, providers, appointments, prescriptions, medical records

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **WADL** - `rest/wadl/`
- **WSDL 1.1 (SOAP)** - `soap/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Patient management
- Provider directory
- Appointment scheduling
- Prescription handling
- Medical records access
