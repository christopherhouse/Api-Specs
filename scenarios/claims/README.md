# Claims Management API

## Description

Insurance — policyholders, policies, claims, documents, adjusters, payouts

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/yaml/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Policyholder management
- Policy administration
- Claims processing
- Document management
- Adjuster assignment
- Payout processing
