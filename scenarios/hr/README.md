# HR Service API

## Description

Employees, departments, leave requests

## Available Specifications

This domain includes API specifications in the following formats:

- **WSDL 1.1 (SOAP)** - `soap/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Employee management
- Department organization
- Leave request processing
- Employee records
