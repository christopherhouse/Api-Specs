# Banking API

## Description

Retail banking — customers, accounts, transactions, beneficiaries, payments

## Available Specifications

This domain includes API specifications in the following formats:

- **OpenAPI 3.0** - `rest/openapi/json/` and `rest/openapi/yaml/`
- **RAML 1.0** - `rest/raml/`
- **GraphQL SDL** - `graphql/`

## API Features

- Customer management
- Account operations
- Transaction processing
- Beneficiary management
- Payment services
